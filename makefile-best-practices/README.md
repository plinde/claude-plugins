# Makefile Best Practices Plugin

A Claude Code plugin that provides best practices for creating self-documenting Makefiles with auto-generated help.

## Overview

This plugin teaches Claude to create Makefiles that:
- Default to showing help when run without arguments
- Auto-generate help text from inline `##` comments

## Installation

```bash
claude plugin install makefile-best-practices@plinde/claude-plugins
```

## Usage

Simply ask Claude to create a Makefile - it will automatically apply self-documenting help patterns.
