# Claude Plugins Repository

## Public Repository Guidelines

**CRITICAL: This is a public repository.**

### Content Restrictions

Never include any of the following in this repository:

1. **Other users' identifying information** - No references to GitHub users other than `plinde`
2. **Credentials or secrets** - No API keys, tokens, passwords, or sensitive configuration
3. **Internal tooling references** - No references to internal-only tools, services, or infrastructure

A pre-commit hook enforces these restrictions automatically.

### Allowed Content

- Copyright notices attributing work to Peter Linde / plinde
- References to `github.com/plinde/` repositories
- Generic, reusable plugin code and documentation
- Public third-party library references

### Example Attribution

```
Copyright (c) Peter Linde
```

When creating plugins, ensure all examples, documentation, and code are generic and suitable for public consumption.
